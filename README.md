# Algorithm Analizer
Program to **analyze** and **compare** different **sorting algorithms**.

Uses `matplotlibcpp` and `Python` to plot graphs.

---
## Features:
- ### Single Mode Analysis
  - **Statistics**
  - **Graph**
- ### Compare Mode Analysis
  - **Graph** for specific **statistic**


 **Statistics** include:
- **Iterations**
- **Swaps** (or writes)
- **Time** (ms)
---
## Notes:
### This program works only with Python installed globally in the system and with the matplotlib and numpy packages installed. If those are missing the program **will not** work. Support for running without graphs and Python import will be added in the future
---
Requirements for **full operation**:
- `Python` installed **globally**
- `matplotlib` package installed **globally**
- `numpy` package installed **globally**

**Minimal operation** without `Python` installation is now **guaranteed** since `v1.1` and as features misses **graphs**
